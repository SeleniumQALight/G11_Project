#001 Test на валідний логін

Precondition:
1. Зареєструвати користувача з унікальним іменем (allash+[date_time])

Steps:
1. В браузері Chrome відкрити login page https://aqa-complexapp.onrender.com.
2. Зробити розмір вікна maximize.
3. Знайти input login, очистити, ввести логін зареєстрованого користувача.
4. Знайти input password, очистити, ввести пароль для цього користувача.
5. Знайти і клікнути кнопку SighIn.

Expected result:
1. Користувач залогінений.
2. Перевірити, що користувач залогінений (як саме):
- наприклад, присутня кнопка SignOut;
- відсутня кнопка SignIn;
- присутня кнопка Create Post.

Postcondition:
1. Видалити щойно створеного користувача.
2. Закрити браузер.


#002 Test на коректність відображення особистої інформації на залогіненій сторінці

Precondition:
1. Зареєструвати користувача з унікальним іменем (allash+[date_time])

Steps:
1. В браузері Chrome відкрити login page https://aqa-complexapp.onrender.com.

3. Знайти input login, очистити, ввести логін зареєстрованого користувача.
